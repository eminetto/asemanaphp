Posts

Bolt - content management (1.6.12) - The Bolt website.

O Bolt é um CMS feito com o micro-framework Silex, que por sua vez é feito com componentes do Symfony. Sou grande fã da simplicidade do Silex e o Bolt segue a mesma ideia, por isso vale a pena dar uma olhada nele

Introducing OctoberCMS - a Laravel-based CMS

O OctoberCMS é um CMS contruído com o Laravel um dos frameworks mais usados da atualidade.

aheinze/cockpit · GitHub

O cockpit possui uma abordagem um pouco diferente para o conceito de CMS. Ele se propõe a ser um "plug & play CMS", ou seja, um componente que pode ser acoplado em outros projetos para fornecer funcionalidades comuns a outros CMS. Realmente uma ideia interessante para projetos que não precisam de toda a complexidade de um CMS completo

duncan3dc/console at 0.1.0 · GitHub

Este projeto é um fork do componente console, do Symfony e fornece algumas funcionalidades extras para quem precisa criar scripts que rodem na linha de comando. Útil para criar regras de negócio que podem ser executadas em background ou agendadas pelo sistema operacional ou outros componentes de um sistema.

kamranahmedse/php-geocode · GitHub

O php-geocode facilita o uso da API de Geocoding do Google . Com ela é possível ter acesso a informações como latitude, longitude, país, cidade, cep e número da rua baseado em um endereço específico.
