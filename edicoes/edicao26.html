Zend Framework 2.4.0 Released! - Zend Framework - Zend Framework

Lançada nova versão do Zend Framework, a 2.4. O interessante é que essa é uma versão LTS (Long-term support), ou seja, que vai ser mantida por bastante tempo. Isso é muito importante para quem está desenvolvendo projetos com o framework pois garante uma maior confiabilidade na escolha do mesmo.
Outro ponto é que agora a equipe vai voltar seus esforços para o desenvolvimento da versão 3.0 que deve sair ainda em 2015
