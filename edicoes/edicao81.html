Links

GitHub - phar-io/phive: phar installation and verification environment

Usar o composer para instalar as dependências dos projetos tornou-se um padrão nos últimos anos, mas esta não é a única forma de fazermos isso. Podemos instalar algumas dependências empacotadas em arquivos .phar e o phive vem para facilitar este processo.

GitHub - producerphp/producer.producer: Tools for producing library packages.

O Producer é uma ferramenta que facilita bastante o dia a dia do projeto. Ele automatiza várias tarefas como analisar a documentação, rodar testes, buscar dados das issues abertas no Github/Bitbucket, validar o composer.json, entre outras.

GitHub - Fakerino/Fakerino: PHP Fake data generator framework with multiple input and output formats.

Como o próprio nome do projeto indica, o Fakerino gera dados falsos e que podem ser usados em testes ou simulações

GitHub - phpthinktank/blast-orm: Framework agnostic data access and persistence based on Doctrine 2 DBAL.

O Blast é um projeto que implementa um novo e mais leve ORM baseado no Doctrine DBAL. Para quem não lembra dos detalhes, o projeto Doctrine é dividido em sub-projetos como o DBAL (Database Abstraction Layer) e o ORM, que faz uso do DBAL. A ideia do Blast é ser uma alternativa ao Doctrine ORM, tarefa bem ambiciosa, na minha opinião

Silex No App Engine | Aprenda tudo sobre Silex e Google App Engine

Video-aulas e tutoriais sobre o Micro-framework Silex e Google App Engine, plataforma de cloud computing da gigante das buscas.
