Posts

Dev Discussions

O dev-discussions foi iniciado por Shawn McCool e é um canal onde desenvolvedores podem discutir assuntos relevantes ao eco-sistema do PHP. Conteúdos como "Command Bus" ou "Active Record vs Datamapper" são exemplos. O canal publica videos e um podcast sobre o assunto e é acompanhado pelo canal #dev-discussions na rede freenode de IRC

Tactician - A simple, flexible command bus

Tactician é uma nova biblioteca focada no padrão Command Bus, padrão este que está em alta na comunidade PHP européia. Este pacote é lançado por Ross Tuck, outro grande nome do PHP e publicado na "League of extraordinary PHP Packages".

phpextra/sorter · GitHub

O pacote phpextra/sorter oferece uma interface OOP para as funções de ordenamento de arrays do PHP. O interessante é que ele oferece uma forma simples de ordenar objetos complexos com regras complexas, usado o array_multisort no plano de fundo.

DoYou.Computer

Lançado por Kayla Daniels, a proposta deste projeto é oferecer desafios mensais de programação, Dojo's virtuais. Cada mês um desafio é feito e ao final todas contribuições são discutidas, o projeto promete ser uma excelente fonte para aprender novas formas de resolver problemas simples.

The Art of Programming by Erika Heidi [Leanpub PDF/iPad/Kindle]

Este é o novo projeto de Erika Heidi, autora do livro "Vagrant Cookbook". Baseado em um post seu de mesmo nome, o livro fala sobre o lado humano da programação e da importância de se ter criatividade e usar o seu "lado humano" na solução de problemas.
