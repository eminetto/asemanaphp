Posts

Under the hood: Box's HHVM migration | Engineering Blog | Facebook Code | Facebook

Neste post podemos ver como o PHP é importante para a empresa Box e como eles tiveram uma melhoria importante de performance ao adotar o HHVM como engine em seus servidores. Artigo interessante tanto pela inovação do uso do HHVM como por provar (se alguém ainda precisa de prova) da seriedade do PHP como linguagem para ambientes complexos.

Lançamento do Zend Server 8.5

Na semana passada a Zend lançou uma nova versão do seu servidor comercial para PHP. A grande novidade é uma maior abertura para a comunidade criar extensões para o excelente componente ZRay, que permite analisar a performance de aplicações PHP

Puli: PHP's Next Package Revolution // Speaker Deck

A ideia do Puli é extender as funcionalidades do Composer adicionando a capacidade de criarmos "módulos" ou "bundles" que funcionem de forma independente entre frameworks. Uma proposta interessante e que merece atenção. Além desta apresentação é possível encontrar mais detalhes no site oficial do projeto

Symfony Certification by Raúl Fraile [Leanpub PDF/iPad/Kindle]

Guia não-oficial para estudo da certificação Symfony. Falando nisso, não esqueça que 2015 é o Ano da Certificação PHP ;)

Experimenting with Broadway — PHP & Symfony

Neste post o autor fala sobre o Broadway, um componente para implementar o pattern CQRS, que pode ser melhor entendido neste post do grande Martin Fowler. Assunto avançado e realmente interessante.
