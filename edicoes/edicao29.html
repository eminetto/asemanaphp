Posts

Introduction to Humbug: A Mutation Testing Framework for PHP | Pádraic Brady

Nessa introdução o autor mostra como usar o Humbug, que é um 'framework de testes mutacionais'. Basicamente ele injeta erros no código para ver se os testes unitários os detectam. É uma abordagem nova ao que já usamos nos nossos testes unitários.

Dica do Jean Carlo Machado

First view on Zend Framework 3 by Matthew O’Phinney

Primeira apresentação sobre as novidades que o Zend Framework 3 vai ter. Apresentado pelo Matthew O’Phinney, líder do projeto na Zend.

Efficient User Timelines in a PHP Application with Neo4j

Post mostrando como usar o PHP para manipular dados no Neo4J. O Neo4J é uma "Graph Database", conceito diferente do que estamos acostumados em bancos de dados relacionais. Neste case é mostrado como criar timelines de eventos a serem mostrados a usuários de forma otimizada.

Horizontally Scaling PHP Applications: A Practical Overview | DigitalOcean

Nesse post a brasileira @erikaheidi mostra como usar técnicas para desenvolver aplicativos PHP que possam facilmente escalar em ambientes de nuvem

Introducing the Symfony Demo application (Symfony Blog)

O pessoal da Sensio Labs criou um projeto de demonstração para quem quer aprender a usar o framework Symfony.
Seria interessante se outros frameworks adotassem esse exemplo e desenvolvessem a mesma aplicação. Assim teríamos o mesmo caso para comparação, o que ajuda no momento de escolher o framework que vamos usar em algum projeto.
