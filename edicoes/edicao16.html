PHP_CodeSniffer 2.2.0 released

Foi lançada uma grande atualização do PHP_CodeSniffer , a versão 2.2.0. O PHP CodeSniffer (phpcs) é uma ferramenta que analisa o código PHP e identifica melhorias no estilo de programação, baseado em padrões como PSR, Zend, etc.
É uma ferramenta de grande importância para manter a qualidade dos códigos que você desenvolve. Na Coderockr nós colocamos um gatilho de pre-commit para validar os códigos e caso o phpcs indique falhas o commit é interrompido.
