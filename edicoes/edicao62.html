PHP DNS Query API Class

A biblioteca é excelente para dar um dig via PHP para lookup DNS. A diferença dessa classe, em relação à algumas funções de network nativas do PHP, é que podemos rodar um dig usando um DNS de Servidor externo, como o Google(8.8.8.8). Muito bom para validar se o DNS já foi migrado/configurado
