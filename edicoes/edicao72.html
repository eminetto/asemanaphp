The RedMonk Programming Language Rankings: January 2016 – tecosystems

O lançamento do PHP 7 parece estar causando uma boa impressão da linguagem. Neste ranking de linguagens, que compara a movimentação do Github, Stack Overflow e outras fontes, o nosso amado PHP aparece em terceiro lugar durante o mês de Janeiro. Ainda é cedo para sabermos se esse comportamento vai se repetir durante o ano, ou mesmo se o ranking é algo válido para comparações, mas é interessante ver esse movimento acontecendo.

Mobile App Develpoments in PHP - PHAP

O PHAP é um projeto ambicioso pois se propõe a permitir a criação de aplicativos mobile nativos a partir de códigos PHP. Parece estar bem no começo mas vale a pena acompanharmos a sua evolução para ver se ele vai conseguir atingir o seu objetivo
