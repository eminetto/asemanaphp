doubleleft/hook · GitHub

Achei este projeto muito promissor. Ele se propõe a ser um "Back-end as a Service (BaaS)". A ideia do projeto é facilitar a criação de back-end para projetos, o que facilita bastante para aplicações baseadas na arquitetura SOA. Eu já usei uma solução comercial para este fim, o Parse e ter uma opção open source e baseada em PHP é uma boa ideia. É um projeto que vale a pena acompanhar, testar e colaborar

The PHP Security Advisories Database

Neste post Fabien Potencier comenta sobre a criação de uma base de dados de principais vulnerabilidades de segurança para PHP. Mostra também como usar a linha de comando do Symfony para fazer a verificação do seu código quanto aos principais problemas de segurança. Também é possível ver como usar esta ferramenta em projetos baseados em outros frameworks. Realmente algo importante para ser usado no dia a dia. Fiquei interessado em integrar isso com uma ferramenta de integração contínua como o Jenkins.
