Splitting the ZF2 Components

Com o avanço do desenvolvimento da versão 3.0 do Zend Framework a equipe passou por uma necessidade complexa: separar cada componente do ZF2 em um repositório próprio, o que é um dos requisitos do projeto do ZF3. Neste post Matthew O'phinney, líder do projeto, mostra como fizeram esta tarefa complicada e que pode ser útil para mais pessoas.

Sylius.org - Modern ecommerce for Symfony2

O Sylius é uma nova solução de ecommerce que está sendo desenvolvida tendo como base os componentes do Symfony2 framework. É um projeto interessante e que merece o acompanhamento de perto pois pode ser uma boa alternativa a grandes players como o Magento.
