A semana: PHP

Essa era para ser uma edição especial, a que iria comemorar o lançamento da versão 7 da linguagem. Mas faltando alguns dias para o lançamento oficial foram descobertos bugs críticos e isso acabou adiando os planos. A equipe do core da linguagem está trabalhando com a nova data de 26/11/2015. Vamos torcer para isso se concretizar, mas de qualquer forma é bom ver o comprometimento de todos com a qualidade. É melhor lançar com algumas semanas de atraso, mas com uma maior qualidade, do que apenas manter a data original.

Lembrando também que a PHPConference Brasil está chegando e que os assinantes da newsletter tem desconto. No ato da sua inscrição, utilize o cupom de desconto semphp (em caso de dúvidas, saiba como usar o cupom) para usufruir do benefício.

pascaldevink/shortuuid - Packagist

Com esta biblioteca podemos gerar UUIDs únicos e fáceis de usarmos em URLs. Usar UUIDs é uma boa prática principalmente quando falamos de APIs. Ao invés de usarmos URLs como /resource/1 usamos algo como /resource/4e52c919-513e-4562-9248-7dd612c6c1ca
Desta forma evitamos identificar a sequência dos IDs usados pelo sistema e aumentamos a segurança

phpbench/phpbench · GitHub

O phpbench é um framework para fazer Benchmarking. Com ele é possível automatizar a execução e geração de relatórios de performance de códigos. Muito útil para identificar gargalos ou mesmo tirar aquela dúvida de performance entre duas ou mais possíveis soluções de um problema
