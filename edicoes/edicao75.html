JPHP - an alternative to PHP

Mais um projeto interessante, levando o PHP para outros caminhos. O JPHP é um compilador que transforma códigos PHP em um bytecode que pode ser executado na JVM do Java. Assim é possível usar classes Java dentro de um código PHP ou mesmo rodar PHP dentro de um ambiente maior, feito em Java.
