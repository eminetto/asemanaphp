Posts

High Performance PHP - PHPNW

Nesta apresentação o autor se aprofunda em aspectos mais "baixo nível" do PH e apresenta formas de melhorar a performance de aplicações. É uma discussão bem aprofundada e interessante para quem precisa usar o máximo dos seus servidores

Configurando SSL no servidor de desenvolvimento (Apache) - PHPit

Neste artigo o grande Rafa Jaques mostra como configurar o SSL em um ambiente de desenvolvimento, algo muito útil para testarmos aplicações que necessitam deste nível de segurança, como e-commerces.

How to Run Multiple Versions of PHP on One Server

Neste artigo o autor mostra como instalar mais de uma versão do PHP no mesmo servidor, junto com o framework Phalcon. É um processo um pouco complexo e que envolve compilação das versões do PHP, mas pode ser bem útil em servidores de testes ou homologação. Em ambientes de desenvolvimento eu prefiro usar o Vagrant, mas para melhorar a performance dos testes pode ser usado o que é demonstrado neste post.

Using Doctrine Migrations as a standalone tool

O Doctrine, ORM mais usado no mundo PHP, possui uma funcionalidade bem interessante chamada Migrations. Com ela é possível programarmos versões da camada de persistência, criando regras para serem executadas em diferentes fases do processo de migração. Neste post o autor mostra como usar esta ferramenta em um ambiente sem frameworks, apesar de usar alguns códigos do Symfony.
