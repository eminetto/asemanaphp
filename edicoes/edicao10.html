Laravel Homestead: ambiente para desenvolvimento PHP | Vedovelli

O amigo Fábio Vedovelli apresenta, em mais um de seus bem produzidos screencasts, o Laravel Homestead. Trata-se de uma máquina virtual para ser usada com o Vagrant e que contém todas as ferramentas necessárias para desenvolvimento de projetos em PHP, com ou sem Laravel.

php.tools by dericofilho

O php.tools é um script que formata o código e o altera para que ele fique de acordo com padrões como o PSR1 e PSR2. É uma opção interessante para ser usado em conjunto com o PHP CodeSniffer que valida o código para verificar se está de acordo com os padrões

Homepage | Skipper

O Skipper é uma ferramenta comercial para a criação da modelagem de dados de projetos PHP, voltados ao uso de ORMs como o Doctrine e o Propel. Ele permite a exportação do diagrama na forma de códigos PHP ou configurações YML, inclusive com geração de código integrado a frameworks como Zend Framework 2 e Symfony. É uma ferramenta multi-plataforma, com versões para Mac, Linux e Windows. Testei a versão trial de 14 dias e me empolguei bastante com o resultado.
