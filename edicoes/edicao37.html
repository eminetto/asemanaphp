Deploying PHP apps to DigitalOcean with Dploy.io

Deploy automatizado é uma técnica muito importante para acelerar o lançamento de funcionalidades ou correções de erros em projetos. Neste post o autor mostra como usar o Dploy.io para fazer deploy de um projeto PHP na Digital Ocean

Zend Server on Azure - Zend Developer Zone

Neste post da Zend vemos como usar o Zend Server na plataforma de cloud computing da Microsoft.

Google Cloud Platform Blog: App Engine for PHP is generally available. Build scalable apps, faster.

Agora o Google App Engine for PHP está oficialmente liberado para todo mundo usar. Nos últimos meses ele era considerado como beta e estava sendo usado por alguns usuários. Com esta liberação esperamos ver mais projetos PHP usando a grande arquitetura de cloud do Google

php.internals: Announcing PHP 5.5 security-only

Um lembrete da equipe de desenvolvimento do PHP da mudança de status do PHP 5.5 que passa a ser atualizado apenas para resolver problemas de segurança
