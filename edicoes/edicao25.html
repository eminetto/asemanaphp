Best PHP Framework for 2015 - SitePoint Survey Results

Resultado da pesquisa feita pelo SitePoint para encontrar os frameworks mais populares de 2015.
Meus 2 centavos sobre a pesquisa. Ela é uma pesquisa de popularidade e não comparação entre "melhores e piores" e deve ser lida desta forma. É complicado comparar frameworks como o Laravel e o ZF2, que são voltados para nichos diferentes e são ótimos em seus objetivos.

https://continuousphp.com/resources/tutorial/create-and-set-up-your-project-on-continuousphp

O ContinuousPHP é uma nova startup que está sendo lançada e promete uma ferramenta para integração contínua voltada a projetos em PHP. Nesse primeiro post eles mostram um passo a passo de como criar um projeto.
É interessante surgirem ferramentas específicas para projetos PHP, sendo que a principal solução para essa tarefa no momento é o Jenkins que é bem genérico e pesado.

guzzle/guzzle at v6 · GitHub

Com a eminente aprovação do PSR-7 já é possível ver alguns projetos fazendo testes e lançando versões preliminares que suportam o padrão.
Nessa branch de desenvolvimento do Guzlle é possível acompanhar o desenvolvimento desta melhoria.
O Guzzle é um cliente HTTP que é muito usado para consumo de APIs e serviços, por isso a adoção do PSR-7 vai ser de grande importância.
